Xnf-tools

Ingo Cyliax (last known email address: cyliax@cs.indiana.edu) developed several tools to support Diglog as a schematic capture environment for Xilinx FPGA chips. For several years, he maintained an FTP site at Indiana University that hosted these tools. Unfortunately, the FTP site is no longer active, and I've lost track of Ingo. If anyone has information on a website that is hosting these tools, please let me know (see email below) and I'll update this page.
